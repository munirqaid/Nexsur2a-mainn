Note:
A simple chat history endpoint is expected at:
GET /api/chat_history?user1=...&user2=...

If you want the server to provide it, add the following route in server.js:

app.get('/api/chat_history', async (req, res) => {
  const { user1, user2 } = req.query;
  const msgs = await Message.find({
    $or: [
      { from: user1, to: user2 },
      { from: user2, to: user1 }
    ]
  }).sort({ time: 1 });
  res.json(msgs);
});

This is easy to paste into server.js above the io handling section if needed.
